# MSI Forged Core

A 480×320 TelemetryForge sample screen built around cyan and magenta hardware
telemetry.

It includes:

- large CPU and GPU usage circles;
- CPU/GPU temperature bars with warning and critical colours;
- RAM, VRAM and Windows volume bars;
- network and GPU power history graphs;
- clock, date, CPU/GPU frequency and GPU power text;
- the bundled `Forged Core` background.

## Import

In TelemetryForge, select **Import package** and open:

`samples/msi-forged-core.telemetryforge`

The package contains the configuration and background image. No personal
paths, hardware-specific fan source or automatic screen rules are included.
