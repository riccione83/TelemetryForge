# Super Widget samples

TelemetryForge bundles these components and screens in release builds:

- **CPU Command Dial**
- **GPU Command Dial**
- **Reactor Core**
- **SDK Hello Dial**

Ready-to-use screen profiles are in [`screens`](screens). External WASM
component packages are also available in [`components`](components) for manual
installation or SDK testing.

The application installs missing samples automatically and never overwrites an
existing screen with the same name.
